About Me

Hello! I am a university student studying in the Computer Engineering program at Marmara University, currently in my third year. I have a keen interest and passion for the field of game development. I had the opportunity to gain valuable experience by completing an internship in this field.

During my internship, I had the chance to learn and apply various aspects of the game development process. I worked on team collaboration, game mechanics and design, visual effects, and programming. This experience has further motivated me to delve deeper into the world of game development.

In addition, I am also honing my skills in web programming. I believe that web technologies have great potential for enhancing user experiences and creating interactive applications on various platforms. I have been practicing web project development using fundamental tools such as HTML, CSS, and JavaScript.

Projects

Face Recognition System
This project is a desktop application developed in Python. It utilizes the advanced technology of dlib, which is based on deep learning, to implement the face recognition feature. Face recognition systems have the ability to detect and recognize faces in an image. In this project, I successfully implemented this feature using Python and the dlib library.

Messaging Application
In this project, I developed a TCP-based messaging desktop application. After running the server, clients can connect to the server by entering the server's IP address and port number. Multiple clients can connect to the server and communicate with each other. This project allowed me to enhance my familiarity with TCP protocols and socket programming, and gain knowledge of how network-based applications operate.

Restaurant Management System
This project is a restaurant management system designed and implemented using the Java programming language. It adheres to object-oriented programming principles and utilizes appropriate data structures. The system supports functions such as order management, inventory tracking, and personnel management for restaurants. Through this project, I gained a better understanding of object-oriented programming in Java and learned about database integration.
